Функции файлов
1) boo_beam_inj(ext).mad - Задание параметров пучка
2) boo_seq_steer_bpm.str - Расстановка элементов по кольцу ускорителя
3) boo_steer_bpm.opt	 - Настройка оптической структуры (задание параметров магнитных полей элементов)
4) error.mad		 - Задание ошибок положения и интегралов магнитного поля дипольных и квадрупольных магнитов
5) quadrupole_matching.mad - Настройка сил квадрупольных магнитов по заданному положению рабочей точки
6) sextupole_matching.mad - Коррекция натуральной хроматичности
7) main.madx		 - Управляющий файл. Вызывает все необходимые подпрограммы и выполняет моделирования.
